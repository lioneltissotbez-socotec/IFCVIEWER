IFC → Plans 2D SAFE

Prototype sans WASM : évite les plantages “non manifold / memory access out of bounds” et extrait les contours 2D depuis le texte IFC.

Cette version ne charge pas la géométrie 3D triangulée. Elle lit directement le fichier IFC STEP et tente d'extraire les profils 2D : dalles, espaces, murs, polylignes, profils extrudés.

Lecture des résultats
• Vert : contours associés à un étage.
• Jaune : géométrie récupérée mais étage incertain.
• Si rien n’apparaît, l’IFC utilise probablement une géométrie BREP/triangulée non exploitable en extraction texte simple.
